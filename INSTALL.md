# Installation Guide
For the prerequisites you can follow the README.md file, once you are done with it, then follow the steps below.
### 1. Standered Installation
```bash
# Install requirements
pip install -r requirements.txt

# Run the App
python app.py
```


### 2. Anaconda (Optional but recommended)
```bash
# Using conda (if you have anaconda)
conda create -n veilarmor python=3.11 -y
conda activate veilarmor

# Install requirements
pip install -r requirements.txt
```
